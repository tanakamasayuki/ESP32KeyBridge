# SwapCtrlCapsLock

キーボードと PC の間に挟んで、CapsLock と Ctrl を入れ替えたいとき用。PC にもキーボードにも設定を入れません。

- 構成: ESP32-P4 1台。USB Host ポートにキーボードを挿し、USB Device ポートを PC へ挿します。
- remap は 1 段の lookup で連鎖しないため、2 エントリの登録がそのまま「入れ替え」になります。長押し・同時押し・ホスト側オートリピートは保たれます。
- 入れ替え後の Ctrl 位置で Caps を切り替えても、PC からの LED report を転送するのでキーボードの LED は正しく追随します。

> **注意**: USB アダプタは現在ビルド確認用のモックです(実装順 7 で実装されます)。
