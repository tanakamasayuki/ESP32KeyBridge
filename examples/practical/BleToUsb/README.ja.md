# BleToUsb

BLE キーボードを USB 有線キーボードとして使いたいとき用。Bluetooth の無い PC、BIOS / リカバリ画面、KVM スイッチ、ペアリングが許可されない環境などで使います。

- 変換なしの素通しです。設定は何も書きません(`addInput` / `addOutput` のみ)。
- PC からの LED report(CapsLock 等)は BLE キーボードへ転送されます。
- スキャン・ペアリング・再接続はアダプタ内の責務です。電池切れなどで接続が失われると、そのキーボードの押下は全 release になります(修飾キー押しっぱなしで切れても stuck key になりません)。

## ハードウェア構成(ESP32-S3)

- ESP32-S3 の USB OTG ポートを PC へ挿します(キーボードデバイスとして見えます)。
- BLE キーボードは ESP32 とペアリングします。

> **注意**: BLE / USB アダプタは現在ビルド確認用のモックです(実装順 7 で実装されます。BLE ライブラリは未定)。
