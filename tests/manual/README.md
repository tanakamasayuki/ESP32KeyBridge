# Manual Tests

Manual checks are used when physical wiring, host OS behavior, browser WebSerial behavior, Bluetooth pairing, or visual confirmation is part of the test.

Examples:

- GPIO matrix wiring.
- Real USB keyboard / barcode reader input.
- USB hub with multiple keyboards.
- USB HID keyboard recognition by a host OS.
- WebSerial reference UI.
- BLE HID pairing and reconnect.
